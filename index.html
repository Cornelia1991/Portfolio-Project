Jungle Booking

Jungle booking is an itinerary building application. The user can create new itineraries by picking a park, an eatery, and an attraction. The app connects to several APIs in order to function. The parks that the user chooses from are fetched from the National Park Services API. Each time a park is selected, the weather forecast for that park is displayed. The weather is fetched from openweathermap.org. The user then needs to select an attraction and an eatery. The data for both of these are fetched from data created by the Nashville Software School. When an itinerary is saved it is posted to a local json database. The data from that local database is fetched, and all the saved itineraries are displayed to the DOM.

Technologies: HTML, CSS, JavaScript, Json

    My Roles
  • Weather section (API Fetch, display on DOM)
  • Style of the main image area
  • Fetch park images from the parks API
  • Assist with overall styling
